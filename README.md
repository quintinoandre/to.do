## Installation

```bash
$ yarn
```

or

```bash
$ npm i
```

## Running the app

```bash
# development
$ yarn start
or
$ npm run start

# watch mode
$ yarn start:dev
or
$ npm run start:dev

# production mode
$ yarn start:prod
or
$ npm run start:prod
```

## Test

```bash
# unit tests
$ yarn test
or
$ npm run test

# e2e tests
$ yarn test:e2e
or
$ npm run test:e2e

# test coverage
$ yarn test:cov
or
$ npm run test:cov
```

## Documentation

After install and running the app, the api documentation can be access through the following url: [http://host:port/api](http://host:port/api)
